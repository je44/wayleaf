First run

Take one minute to understand Wayleaf

Wayleaf turns your new tab into a local-first workspace. These notes help you start safely and quickly.

Local first

History, bookmarks, shortcuts, and theme settings stay in Chrome extension storage. Wayleaf has no backend account.

Permissions power the page

history powers recent browsing, bookmarks powers bookmark folders, and tabs plus scripting help open search results and AI pages.

Settings try to follow Chrome sync

The same Google account restores preferences automatically. If sync is unavailable, settings stay on this device.

AI commands have a fallback

Use /gpt, /claude, /gemini, /grok, /deepseek, /doubao, /kimi, or /glm to open the AI page and try to fill your prompt. If the site requires login or changes, paste the saved prompt manually.

Start with two actions

Add one favorite site, then choose a bookmark folder from the navigation hub. Theme and sync stay in Settings.

Report issues directly

# tab-tab Chrome 扩展

`tab-tab` 会替代 Chrome 新标签页，提供三栏布局：

- 左侧：热门门户网站入口，默认站点优先使用本地高清 SVG 图标，动态站点回退到 Chrome favicon 服务。
- 左侧入口支持自定义添加和删除，数据保存在本机扩展存储中。
- 中间：用户选择一个 Chrome 书签文件夹后，按热门入口样式展示该文件夹里的网页书签；文件夹选择在二级面板中完成。
- 右侧：最近一周浏览历史按网站归组，组头展示网站图标和网站名，组内列出最近页面标题，并支持将页面置顶。

## 安装

1. 打开 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目目录，例如 `tab-tab-chrome-extension`

## 权限说明

- `bookmarks`：读取书签文件夹和书签项，用于中间书签文件夹面板。
- `history`：读取最近历史记录，用于右侧浏览记录网格。
- `favicon`：通过 Chrome 官方 favicon API 显示动态历史记录的网站图标。
- `storage`：保存用户自定义添加的热门入口、已选书签文件夹和置顶历史项。

扩展不包含远程脚本，也不会上传浏览器数据。

默认入口图标来源记录见 `docs/icon-sources.md`。
